Frame Module¶

The Frame module provides a class with the same name which is used to represent a CPS2 Frame. This module has a few factory functions and provides functions for converting the Frame to a variety of formats for further processing.

Example¶

A Frame acquired by using gather and then printed with to_png():

The Frame Class¶

class cps2zmq.process.Frame.Frame(fnumber, sprites, palettes)¶

Frame is a container of Sprites. It is related to the frames drawn by the CPS2 hardware. Ideally this would provide methods for manipulating sprites en masse.

fnumber¶

int – the frame number. Frames should have a unique fnumber (linearly increasing).

sprites¶

list of Sprite.Sprite – the Sprites of the Frame

palettes¶

dict – contains the 32 palettes used by Sprites/ColorTiles in the Frame

The Frame class has the following methods

Frame.add_sprites(sprites)¶

Add sprites to the Frame’s existing sprites.

Parameters:sprites (list of Sprite.Sprite) – a list of sprites
Frame.to_png(fname, imsize=(400, 400))¶

Saves the Frame as a PNG file.

Parameters:
  • fname (str) – the name to save the PNG file as.
  • imsize (int, int, optional) – The size of the PNG file. Defaults to (400, 400)
Frame.to_file(path)¶

Saves the Frame object as a json encoded file.

Parameters:path (str) – The location to save to.
classmethod Frame.from_file(path)¶

Returns a Frame from a json encoded file.

Parameters:path (str) – path to the json file.
Returns:a Frame object
