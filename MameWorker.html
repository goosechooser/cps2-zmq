MameWorker Module¶

The MameWorker module contains a class of the same name. Functionality related to processing of message contents are done here.

Examples¶

The MameWorker Class¶

class cps2zmq.gather.MameWorker.MameWorker(idn, front_addr, service, pub_addr=None, log_to_file=False)¶

MameWorker turns data into Frames.

w_id¶

str – The worker’s ‘id’. This an address that is appended to the front of messages it sends to the server.

context¶

zmq.Context – required by ZMQ to make the magic happen.

The other side of the socket is usually bound by a MameServer.
socket_addr¶

str – The address to connect the frontend socket to. This is usually set

frontend¶

zmq.Context.socket – Socket that connects to the MameServer address. Sends requests for work.

working¶

bool – Whether the worker is in its main loop or closing.
